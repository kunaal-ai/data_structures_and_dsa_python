   - Common patterns: Pair sum, removing duplicates, in-place operations

2. **Sliding Window**
   - When to use: Subarray/substring problems with fixed or dynamic window size
   - Common patterns: Maximum sum subarray, longest substring with K distinct characters

3. **Prefix Sums**
   - When to use: Range sum queries, subarray sum problems
   - Common patterns: Contiguous subarray sum, equilibrium index

4. **In-place Operations**
   - When to use: When O(1) space is required
   - Common patterns: Array deduplication, move zeros, in-place reversal
