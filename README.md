# 12-Week DSA & Coding Mastery Plan

A structured learning path to master Data Structures and Algorithms, with a focus on interview preparation and practical problem-solving.

## üìÖ Weekly Breakdown

### Week 1: Arrays, Strings & Analysis
- **Focus**: Two-Pointer, Sliding Window, Prefix Sums, In-place operations
- **Key Concepts**: Time/Space complexity analysis, recognizing patterns
- **Practice Goal**: 15 problems focusing on pattern recognition
- **Directory**: `week_01_arrays_strings/`

### Week 2: Hash Maps, Sets & Sorting
- **Focus**: Frequency counting, anagram grouping, custom key design
- **Key Concepts**: Merge Sort, Quick Sort, hash map optimizations
- **Practice Goal**: 10-12 problems connecting hash maps to sorting
- **Directory**: `week_02_hashmaps_sets_sorting/`

### Week 3: Stacks, Queues & OOD Basics
- **Focus**: Monotonic stacks, queue-based BFS, OOD principles
- **Key Concepts**: LLD (Low-Level Design) basics, stack/queue implementations
- **Practice Goal**: 10 problems + 1 OOD design
- **Directory**: `week_03_stacks_queues_ood/`

### Week 4: Trees & Binary Search
- **Focus**: Tree traversals, BST operations, Binary Search variations
- **Key Concepts**: Recursion, tree properties, search space reduction
- **Practice Goal**: 15 problems focusing on BST and Binary Search
- **Directory**: `week_04_trees_binary_search/`

### Week 5: Graphs Fundamentals & Search
- **Focus**: BFS/DFS, path finding, cycle detection
- **Key Concepts**: Graph representations, Dijkstra's algorithm, Union-Find
- **Practice Goal**: 12 graph-based problems
- **Directory**: `week_05_graphs_search/`

### Week 6: Heaps, Tries & Advanced Strings
- **Focus**: Top K elements, trie implementations, string manipulation
- **Key Concepts**: Heap properties, trie operations, advanced string matching
- **Practice Goal**: 10 problems including "Merge K Sorted Lists", "Word Search II"
- **Directory**: `week_06_heaps_tries_strings/`

### Week 7: Dynamic Programming I
- **Focus**: 1D DP, memoization, tabulation
- **Key Concepts**: State transitions, overlapping subproblems
- **Practice Goal**: 10 classic DP problems (climbing stairs, coin change, etc.)
- **Directory**: `week_07_dp_1d/`

### Week 8: Dynamic Programming II & Review
- **Focus**: 2D DP, 0/1 Knapsack, LCS patterns
- **Key Concepts**: 2D state transitions, space optimization
- **Practice Goal**: 12 mixed DP and array/string problems
- **Directory**: `week_08_dp_2d_review/`

### Week 9: Advanced Graphs & Bit Manipulation
- **Focus**: Topological sort, advanced Union-Find, bit operations
- **Key Concepts**: Graph algorithms, bit manipulation techniques
- **Practice Goal**: 8-10 problems including "Course Schedule II", "Number of 1 Bits"
- **Directory**: `week_09_advanced_graphs_bits/`

### Week 10: Recursion, Backtracking & Math
- **Focus**: Subsets, permutations, combinatorial problems
- **Key Concepts**: Recursive thinking, backtracking patterns, math fundamentals
- **Practice Goal**: 10 problems with clean recursive implementations
- **Directory**: `week_10_recursion_math/`

### Week 11: System Design Fundamentals
- **Focus**: Scalability, load balancing, caching strategies
- **Key Concepts**: Distributed systems, database design, API design
- **Practice Goal**: 5 system design problems
- **Directory**: `week_11_system_design/`

### Week 12: Mock Interviews & Final Review
- **Focus**: Full problem-solving sessions, time management
- **Key Concepts**: Interview strategies, problem decomposition
- **Practice Goal**: 5+ mock interviews, comprehensive review
- **Directory**: `week_12_mock_interviews/`

## üöÄ Getting Started

### Prerequisites
- Python 3.8+
- pip (Python package manager)

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/data_structures_and_dsa_python.git
   cd data_structures_and_dsa_python
   ```

2. Create and activate a virtual environment (recommended):
   ```bash
   python -m venv venv
   source venv/bin/activate  # On Windows: .\venv\Scripts\activate
   ```

3. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
